# analytics-capstone-satyakala

**Title:** End-to-End Sales Analytics Pipeline  
**Author:** Satyakala Devata

**Short project summary:**  
End-to-end analytics capstone demonstrating data ingestion, cleaning, transformation, modeling and visualization using Python, SQL and Tableau/Power BI. Designed to showcase skills for Senior Data Analyst / Analytics Engineer roles.

**Quick start:**  
- Open `/notebooks/01_data_loading_and_eda.ipynb` to see initial data loading & EDA.  
- Follow `/notebooks/02_data_cleaning_and_feature_engineering.ipynb` (coming) for cleaning & features.  
- Final dashboard will be in `/dashboard/`.

**Tools / Stack:** Python (pandas), SQL, GitHub, Google Colab, Tableau / Power BI
